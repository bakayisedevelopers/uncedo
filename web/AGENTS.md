# Web App Agent Instructions

Read this file after the repository root `AGENTS.md` when working in `web/`.

## What To Read

1. `../AGENTS.md`
2. `README.md`
3. `../docs/skills/agent-codebase-guide.md`
4. `../docs/skills/agent-developer-workflow.md` when deployment or release steps are involved

## Scope

- `web/` is the React + Vite web application.
- Keep route, service, shell, and policy changes reflected in `web/README.md`.
- Preserve the current web app structure unless the task explicitly changes it.

## Change Rules

- Update the web README when routes, pages, services, or environment requirements change.
- Keep edits within the web app unless the request explicitly spans other packages.
