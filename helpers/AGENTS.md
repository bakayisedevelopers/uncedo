# Helpers App Agent Instructions

Read this file after the repository root `AGENTS.md` when working in `helpers/`.

## What To Read

1. `../AGENTS.md`
2. `README.md`
3. `../docs/skills/agent-codebase-guide.md`
4. `../docs/skills/agent-developer-workflow.md` when builds or release APKs are involved

## Scope

- `helpers/` is the helper Expo React Native app.
- Keep helper workflows aligned with the helper screens and services in this folder.
- Reflect significant flow or structure changes in `helpers/README.md`.

## Change Rules

- Update the README when screens, services, navigation, or build steps change.
- Keep edits scoped to the helper app unless the request explicitly spans other packages.
